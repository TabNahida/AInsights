Methodology

AInsights Index 计算方式

AIndex 是 AInsights Index 的别名，两个名称指向同一套默认排名口径。

AInsights Index / AIndex

AInsights Index 使用 Artificial Analysis Intelligence Index evaluation suite 的公开测试项权重重新计算综合分。AA-Omniscience 的 12.5% 权重全部计入 Accuracy，Non-Hallucination Rate 权重为 0。

Geometric Weighted Mean

默认综合分使用 Geometric Weighted Mean。AIndex 先把每个测试项转换为 Best score ratio：模型分数 / 该测试项最高分，再进入几何加权均值。

Best score ratio

相对比例聚合后会乘回全站最高 AA Intelligence Index 分数，作为最终展示分。因此每个测试项先在自己的最高分基线上比较，再映射回 AA Intelligence 的分数尺度。

缺失和 0 分

默认 AInsights Index 会把缺失项作为 0 比例计入总权重。因为公式使用 ratio + 1，真实 0 分或缺失项会显著降低结果，但不会让整个模型的综合分直接归零。

Custom Weight

Custom weights 面板可以独立切换 Best score ratio / Raw score 和 Geometric Weight Mean / Weight Mean，并继续使用缺失扣分强度、最低覆盖率和逐项权重滑块做对照。

数据来源：Artificial Analysis。 AInsights Index / AIndex 基于其公开评测数据重新计算。
